# Qotd

Displays a new quote upon every login to a Terminal.

## Todo

## Installation

Add this line to your application's Gemfile:

gem 'qotd'

And then execute:

$ bundle

Or install it yourself as:

$ sudo gem install qotd   # Sudo may not be needed.

## Usage

Make it run in a bash config file.

# in ~/.bashrc
ruby qotd
